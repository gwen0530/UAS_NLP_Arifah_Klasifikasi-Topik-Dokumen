Tentang Aplikasi

Document Topic Classification System adalah aplikasi web berbasis Machine Learning yang dirancang khusus untuk mengklasifikasikan topik dokumen teks Bahasa Indonesia secara otomatis. Sistem ini memanfaatkan algoritma Logistic Regression yang dipadukan dengan teknik TF-IDF Vectorization serta pipeline preprocessing NLP lengkap menggunakan library Sastrawi untuk memproses teks Bahasa Indonesia dengan akurasi tinggi.

Aplikasi ini dibangun sebagai proyek akhir semester Mata Kuliah Natural Language Processing (NLP), dengan antarmuka profesional berbasis Flask, Bootstrap 5, dan Chart.js.

Alur Sistem NLP Pipeline
1
Input Dokumen Teks Bahasa Indonesia
User memasukkan teks dokumen atau sistem membaca dari dataset CSV.
2
Text Cleaning & Normalisasi
Konversi ke lowercase, hapus HTML, angka, simbol, tanda baca, dan whitespace ganda.
3
Tokenizing
Pemisahan teks menjadi urutan token kata individual.
4
Stopword Removal (Sastrawi)
Pembuangan kata umum / kata hubung yang tidak informatif dalam Bahasa Indonesia.
5
Stemming Bahasa Indonesia (Sastrawi)
Pengubahan kata berimbuhan menjadi kata dasar menggunakan algoritma Enhanced Confix Stripping (ECS).
6
TF-IDF Vectorization
Transformasi teks menjadi vektor fitur numerik dengan pembobotan TF-IDF (max_features=2500, ngram_range=(1,2)).
7
Logistic Regression Classification
Prediksi topik dokumen dengan skor probabilitas untuk setiap kelas topik.

Cara Penggunaan
  1. Upload Dataset: Buka halaman Dataset dan upload file CSV dengan kolom text dan label.
  2. Training Model: Buka halaman Training Model dan klik tombol "Mulai Training Model" untuk melatih model klasifikasi.
  3. Testing / Prediksi: Buka halaman Testing, masukkan teks dokumen Bahasa Indonesia dan klik tombol "Prediksi Topik Dokumen".
  4. Evaluasi: Lihat performa model (Accuracy, Precision, Recall, F1-Score, Confusion Matrix) di halaman Evaluasi.
  5. Visualisasi: Eksplorasi grafik distribusi data, word frequency, dan riwayat prediksi di halaman Visualisasi.
  6. Export: Download laporan evaluasi dalam format PDF atau riwayat prediksi dalam format Excel.
